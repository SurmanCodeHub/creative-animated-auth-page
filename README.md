# 💻 Animated Login & Sign-Up Page

💻 Live Interactive Login/SignUp page with smooth custom animations. Built using HTML, CSS, and JavaScript. Optimized for desktop/laptop views.

---

## ✨ Features
- 🔄 Smooth switching animation between Login and Sign-Up forms.
- 💻 Optimized for Laptop and Desktop screens.
- 🎨 Modern UI with clean glassmorphism and gradient effects.
- ⚡ Lightweight and fast-loading.

## 🛠️ Tech Stack Used
- **HTML5** - For structured layout.
- **CSS3** - For styling, custom animations, and layout.
- **JavaScript** - For form toggling and interactive elements.

## 🚀 How to Run Locally
1. Clone this repository or download the ZIP file.
2. Open the `index.html` file in any modern web browser (Chrome, Edge, Brave, etc.).
